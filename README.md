# Titanic Dataset Exploratory Data Analysis (EDA)

## About the EDA

This focuses on performing Exploratory Data Analysis (EDA) on the Titanic dataset. The main goal is to understand the dataset using statistics and visualizations before applying any Machine Learning techniques.

EDA helps in identifying patterns, trends, missing values, and relationships between different features in the dataset.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset

The Titanic dataset contains information about passengers such as age, gender, ticket class, fare, and survival status.

## Tasks Performed

### Data Loading
- Imported the dataset using Pandas.
- Displayed the first few rows to understand the data structure.

### Data Exploration
- Checked the number of rows and columns.
- Viewed data types of each feature.
- Examined missing values present in the dataset.

### Statistical Analysis
- Generated summary statistics such as mean, median, minimum, maximum, and standard deviation.

### Data Visualization
Created different visualizations to better understand the dataset:

- Histogram for Age distribution
- Boxplot for Fare analysis
- Correlation Heatmap
- Pairplot for feature relationships

## Key Observations

- The dataset contains 891 passenger records.
- Some columns such as Age, Cabin, and Embarked contain missing values.
- Most passengers were between 20 and 40 years old.
- The Fare column contains several outliers.
- Female passengers had a higher survival rate compared to male passengers.
- Passenger class appeared to influence survival chances.

## Conclusion

Through this EDA process, I gained a better understanding of the Titanic dataset and learned how visualizations and statistical analysis can help uncover useful insights from data. This analysis also highlights the importance of data preprocessing before building Machine Learning models.
